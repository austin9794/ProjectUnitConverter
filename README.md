## A simple Java-based unit converter that converts:
- Celsius ↔ Fahrenheit
- Kilograms ↔ Pounds

## Concepts Used
- Java methods
- Conditional statements
- User input 
- Clean package structure

## How to Run
1. Compile the program
2. Run `UnitConverter.java`
3. Follow the menu prompts
